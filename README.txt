PWA "Calcolo Orari" con icona IRIS Web

Carica TUTTI i file nella root del repository (GitHub Pages):
- index.html
- manifest.webmanifest
- sw.js
- /icons (3 PNG)

Poi su iPhone:
- Elimina l'icona precedente dalla Home (se presente)
- Safari → apri l'URL → Condividi → Aggiungi a Home
- Se non cambia: Impostazioni → Safari → Avanzate → Dati siti web → elimina per github.io
